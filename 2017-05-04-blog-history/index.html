博客搭建日志

发表于 2017-05-04

一. 2017年5月4日 迁移博客至新电脑(台式机)
操作：

  1. 新电脑安装Git Shell以及node.js(依照hexo官网);

  2. hexo初始化及next主题获取(依照各自网站操作);

  3. 旧博客重要文件迁移至新博客
    1) 站点配置文件 _config.yml 完整替换初始化hexo站点配置文件;
    2) source下_post文件夹下旧文章;
    3) source下.nojekyll & CNAME 文件(作用:配置个性域名)
    4) 旧博客next/source/images下ico文件

  4. 主题配置文件(参照旧博客文件按需按新版本修改)
    1) menu 设置;
    2) 在language下zh-Hans文件自定义新增的menu对应中文命名;
    3) 开启next另一种主题模式Pisces;
    4) social部分的修改;
    5) social_icons部分的修改;
    6) 修改版式风格sidebar为right,默认为left;
    7) 修改代码模式显示highlight_theme为night eighties,默认为normal;
    8) 保留新版本的vendors: _internal: lib.这部分是为了能正常push到github并正常显示博客;
    9) 在配置文件最后新增语句：# like © 2013 - 2015 since: 2016 显示博客起止年份

二. 修改文章页面的上一页、下一页位置(位置互换)

  1. 修改位置: next/layout/_macro/post.swig 查找previous或next即可找到位置修改

三. 站点文章显示及访问优化

  1. 修改 <!--more--> 效果，点击“阅读更多”进入文章不自动定位到阅读位置。修改位置：scroll_to_more: false # default: true
  2. 实现文章首页隐藏以及全面隐藏。修改位置：next/layout下index.swig和archive.swig文件下述关键地方增加 visible !== 'hide' 语句。注：目前修改为文章全面隐藏，部分隐藏待完善

存在问题：上一页、下一页仍存在该文章名

待完善，实现文章部分隐藏
