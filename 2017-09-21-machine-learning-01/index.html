机器学习笔记一：概述

发表于 2017-09-21 | 分类于 机器学习

机器学习笔记

记录机器学习的一点一滴，笔记主要内容来自Course上Andrew Ng教授Machine Learning课程，间附加了一些网络内容综合

定义

机器学习的定义，Andrew Ng教授阐述了两种定义：

the field of study that gives computers the ability to learn without being explicitly programmed. ——Arthur Samuel

可译为：让计算机在没有明确编程的情况下，获得学习能力的领域。

A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E. ——Tom Mitchell

可译为：计算机程序从一系列以（P）为结果的任务（T）中得到经验（E），在大量经验（E）的累积下，最终表现的结果（P）得以改善。

其中E为已明确的数据集，T为任务目标，P为需要预测的结果值。根据已有的数据集E，去预测得到结果P，来完成任务T，这就是一个机器学习的过程。

一般来说，机器学习可主要分为两类：监督学习和无监督学习。

几种学习类别

监督学习

所谓“监督学习”，是指输入和输出之间存在一定关系，并且我们对于输出的结果有一个明确的期望（了解）。采用术语来描述则为：“监督式学习算法包括一个目标变量（因变量）和用来预测目标变量的预测变量（自变量）。通过这些变量我们可以搭建一个模型，从而对于一个已知的预测变量值，我们可以得到对应的目标变量值。重复训练这个模型，直到它能在训练数据集上达到预定的准确度。 ”

其中”监督学习“又可分为两类，分类（Classification）与回归（Regression）问题，又可称之为离散型问题和连续型问题。

分类问题中输出的结果是离散的，此类问题的结果只有0或1两种结果，其将输入变量映射到离散类别；回归问题中输出结果是连续的，其可以包含多种结果，其将输入变量映射到一些连续函数。

如，已有肿瘤良性恶性的数据集，通过一个要判断的肿瘤大小来判断这个肿瘤是良性还是恶性，是一个分类（离散型）问题；已有一部分房子真实的数据，通过一个要判断房子的面积来预测房子的价格，是一个回归（连续型）问题。

所以，如果我们通过监督学习的方式，利用股票的历史数据，来预测股票的未来价格，其实就是要解决连续型问题；而利用股票的历史数据，来判断股价下一次是涨还是跌，就是要解决离散型问题。

监督学习代表算法模型有：

- 回归模型
- 决策树
- 随机森林
- KNN邻近算法
- 逻辑回归

无监督学习

所谓*“无监督学习”，是指对于输出的结果不了解，没有一个明确的概念。同时，不清楚变量之间的相互影响关系，其根据某一指标来进行聚类，最后表现出一种结构性特征，预测结果得不到反馈。“在无监督学习中我们没有需要预测或估计的目标变量。无监督式学习是用来对总体对象进行分类的。它在根据某一指标将客户分类上有广泛应用。”

“无监督学习”也有聚类和非聚类之分，聚类问题则可以得出相关结构性特征，而非聚类问题，则可能会得到一种非相关性结构（如鸡尾酒会算法）。

如，我们可以收集100万个DNA数据，并基于某些规则如寿命、位置、作用等将这些DNA组成相关的结构组，属于聚类问题；对一个鸡尾酒会声乐音频，将其人声和音乐分别区分出来（鸡尾酒会算法），属于非聚类问题

无监督学习代表算法模型有：

- 关联规则
- K-means聚类算法

强化学习

实际上，除了“监督学习”和“无监督学习”之外，还有一种学习类别称为“强化学习”。

强化学习训练程序做出某一决定，在某种情况下程序尝试所有的可能行动，记录不同的结果并找出最好的一次尝试来做决定。

强化学习的代表算法模型为马尔科夫决策过程。

机器学习算法

目前常用的机器学习算法主要包含以下几类：
1.线性回归 (Linear Regression)
2.逻辑回归 (Logistic Regression)
3.决策树 (Decision Tree)
4.支持向量机（SVM）
5.朴素贝叶斯 (Naive Bayes)
6.K邻近算法（KNN）
7.K-均值算法（K-means）
8.随机森林 (Random Forest)
9.降低维度算法（Dimensionality Reduction Algorithms）
10.Gradient Boost和Adaboost算法

相关算法可以参考机器学习算法一览（附python和R代码），也可以自行了解其具体含义。
