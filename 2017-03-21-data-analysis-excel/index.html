数据分析之Excel养成

发表于 2017-03-21 | 分类于 数据分析

Excel技能主要从以下三个方面讲述：

  • 各类快捷键技巧
  • 常用函数使用
  • 能够通过搜索解决问题

VBA暂不在讲述范围内。

操作技巧：

  1. 将常用功能加入自定义功能区，快速调用；

    在“自定义快速访问工具栏”添加常用Excel命令，如粘贴数值、粘贴并转置、插入数据透视表等功能，可采用“Alt+数字”快速调用该功能（数字表示在快速访问工具栏的次序）
    如何添加？可通过“开始-选项-快速访问工具栏”实现，或者工具栏“撤销/恢复”右侧下拉按钮-“其他命令”唤出。添加的功能点包括：常用命令、不在功能区的命令、宏、各选项卡选项等

  2. 去掉excel默认网格线：“视图”-“网格线”（显示栏中）

  3. 冻结固定行/列：“视图”-“冻结窗格”-“冻结拆分窗格/冻结首行”/“冻结首列”

  4. 删除重复值：“数据”-“删除重复项”（数据工具栏）

  5. 标记重复值：“开始”-“条件格式”-“突出显示单元格规则”-“重复值” （可根据需要设定标记的格式）

  6. 根据一定条件标记符合条件的单元格：“开始”-“条件格式”-“突出显示单元格规则”/“项目选取规则”

    应用场景：对某列中小于70%的数值标记为红色数值

  7. 单元格数据可视化：“开始”-“条件格式”-“数据条”/“色阶”/“图标集”

  8. 按一定规则将单元格内容拆分：“数据”-“分列”

    应用场景：可对某列单元格按“_”拆分，也可对某列单元格按“市”拆分，获取市前面的文字

单元格处理

  1. 快速删除/增加单元格/行列：删除：“Ctrl” + “-”，增加：“Ctrl” + “Shift” + “+”

    注：
    1) 可应用于单元格的增减和行列的增减；
    2) 快速增加行列准确理解应为为“快速插入”，可用于复制单元格/数值的插入，此时可选择选中单元格“下移”或“右移”。

  2. 批量格式刷：双击格式刷

    注：
    1) 双击格式刷完成所需要区域的处理后，需要“Esc”键退出格式刷模式，否则将继续应用格式刷；
    2) 双击格式刷，处理的区域与格式刷选中的第一个单元格有关，横向批量格式刷，处理到第一个单元格所在行最后一个有内容列，竖向批量格式刷，第一个单元格所在列之后仍有数值，处理到最后一个内容对应的行，第一个单元格对应列之后没数值，批量选中的excel所有行（all，百万行）。

  3. 快速切换不同工作表sheet：“Ctrl” + “Pgup”/“PgDn”

  4. 快速定位光标所处单元格：“Ctrl” + “↑/↓/←/→”

    应用场景：
    1) 快速定位到所需单元格；
    ​ 2) 某一特定区域单元格的快速选取/全选，选中第一个单元格，并执行 Ctrl+↑/↓/←/→ 完成该区域单元格选中

  5. 选中多个单元格：“Ctrl” + 鼠标点选 或 “Shift” + “↑/↓/←/→ ”

    注1：Ctrl与Shift对于单元格处理有四种不同的功能效果

    功能：
    1) “Ctrl” + “↑/↓/←/→”：快速定位到该列/该行的首尾单元格；
    2) “Ctrl” + “鼠标点选”：快速同时选择多个单元格（对整行/整列同样适用）；
    3) “Shift” + “↑/↓/←/→”：逐列/逐行同时选中下一单元格；
    4) “Shift” + “鼠标点选”：全部选取原单元格与点选单元格之间的所有单元格（对整行/整列同样适用）

    注2：功能2、3也同样适用于sheet表格选取

    注3：在多个连续单元格、行/列、sheet工作表的选取中，应用Shift更具便捷性

  6. 根据文字内容调整行列宽度：移到行列的边缘，双击（即可实现自动调整）

    注：也可拖拉法，拖拉法需手动调整列/行宽高

  7. 单元格内换行：“Alt” + “enter键”

数据处理

  1. 快速求和：“Alt” + “=”

    功能等于使用“=SUM()。另：在需要使用“=SUM()”函数又不想直接输入公式时也可使用该快捷键

  2. 快速批量应用函数：双击或拖拽

    应用场景：
    1) 批量使用函数求值;
    2) 快速填充序列/相同数值快速填充。如该列全为1，可快速“双击/拖拽”实现，该列为序号“1，2，…，10”，可填写“1,2”后“双击/拖拽”实现

    备注1：
    1) 双击右下角小黑点，此时鼠标样式为黑色十字；
    2) 对“列”批量应用采用“双击”，对“行”批量应用采用“拖拽”；
    3) 列双击处理时，需保证处理数据列前一列/后一列也有数据，否则只能采用“拖拽”小黑点处理。双击处理数据至前一列/后一列数据末尾
    4) 该功能可实现“复制单元格”、“仅填充格式”、“不带格式填充”三种功能，对数字来说还可实现“填充序列”功能。以上四种功能需要点击右下出现的小图标进行功能选择

    备注2：
    1) “复制单元格”功能指同时应用单元格格式与函数（新单元格对应的函数）；
    2) “仅填充格式”功能指保持原单元格的数值不变，仅应用单元格格式（双击/拖拽处理后，原单元格数据被覆盖一次，该功能会复原数据）；
    3) “不带格式填充”功能指保持应用单元格函数，但不应用单元格格式；
    4) “填充序列”功能保持序列+1格式，也可以直接输入“1,2”然后“双击/拖拽”实现该功能

  3. 相对引用/绝对引用：F4，多次选择可单独锁某行/某列

  4. 根据条件快速定位特定类内容：F5

    应用场景（常用）：
    1) 某一区域只选择有数值的单元格，用于函数批量使用，主要区别于空单元格；
    2) 某一区域只选择空单元格，用于对空单元格内容快速输入（要求空单元格内容与其所在上一单元格内容相同可应用“Ctrl” + “Enter”实现）；
    3) 其他内容参见定位功能可选择的条件和功能

  5. 快速变成百分比：“Ctrl” + “Shift” + “5”

    注：该功能也可直接直接点击菜单工具栏的“百分比样式”应用

    注2：菜单工具栏的快捷方式等应常使用，如转换成百分数，应用千分位，，增/减一位小数等

  6. 一键展示公式：“Ctrl” + “`”，再次使用复原

    使用场景：检查一篇区域公式正确与否

函数

  1. IF系列

    1) IF: =IF(表达式，条件为真时结果，条件为假时结果)
    2) IFERROR: =IFERROR(表达式，表达式错误返回结果) 两种返回结果：表达式成立，返回表达式值；反之则显示第二个参数“表达式错误返回结果”

  2. SUM系列

    1) SUM: =SUM()
    2) SUMIF: =SUMIF(选取范围,需要满足的条件) ，=SUMIF(选取范围,需要满足的条件,真正计算求和的对应条件区域)

  3. Trim() 清除单元格空格（仅限全部单元格内容的最前面或者最后面）

    Clean() 可清楚多余空行等（alt + enter 造成的空行等）

NETWORKDAYS和WORKDAY函数

NETWORKDAYS(A,B)

WORKDAY(A,days)

注：不包含周末和专门指定的假期

index + match 配合查找某类值的所在区域并给出结果

明确知道要查找的具体行列位置，“=INDEX(A2:D10,2,3)”，即“查找/显示”A2-D10区域内，第2行第3列数值

不采用具体行列，通过MATCH语法查找定位。“G4=INDEX(A1:D10,MATCH(F2,A1:A10,0),MATCH(F4,A1:D1,0))”，即G4区域显示的是A1-D10区域内，查找同时匹配A1-A10中带F2关键词和A1-D1内带F4关键词的内容值

数据可视化

  1. “开始”-“条件格式”-“突出显示单元格规则”/“项目选取规则”

  2. “开始”-“条件格式”-“数据条”/“色阶”/“图标集”

  3. “开始”-“套用表格样式”
