每日随想之明确自己的方向

发表于 2017-07-02 | 分类于 随笔

嗯，这篇文章也是今天所要表达内容的一个产物。具体如下：

2017年7月1日，2017年下半年的第一天，引起感想的主要有两个事情：一个是看到了Richor同学（离线上海粉丝群成员）的个人网站，里面属于她自己的photography作品和design作品给了自己触动，另一个是参加Ping++举办的“Growth Workshop：企业如何用数据驱动GMV增长”所引发的一些思考。而这二者，暴露了自己很严重的几个问题。

Richor同学的个人网站主要分为四个部分，Blog/Photography/Design/About

第一部分Blog主要就是个人博客，这点对自己并没有很大触动，最主要触动自己的事Photography和Design两个。平时就有看到她有在拍摄一些内容，街景或者人物，在朋友圈看到只是一种感慨和佩服和羡慕她有这种想法和行为，而在网站看到，则说明她在自我记录，并且尝试在photography这条路上走得更远（当然有可能并非是photography但至少其内容体现是不变的），对于内容的追求、记录和展现，她一直都有在做，并非只是一时的兴趣而没有长久的明确的一个规划和想法，同样能体现的是Design部分，里面是她的一些作品。（记得与她见面时就得知她是一个爱好设计的复旦研究生（和利器群刘佳花一样），之后有去日本进修的想法） 正如她在About里描述的那样，All she desires is to create and curate about culture visually，并且从此之外，她还介绍她自己是一个freelancer这点对自己的触动是最大的。这体现了她对于自己的未来和自己有着清晰的认识和定位，并且对于自己的未来规划有着实际的行动（Photography和Design）,而这正是我所欠缺的。对于自己的定位和认识，无论是freelancer还是“to create and curate about culture visually”，都值得自己去好好反思并且找到自己对于自己正确的定位以及之后的成长目标。

第二个触动则是一场关于Growth的讲座。老实说，对自己对这方面的知识贡献和业务指导作用有限（当然自己没有认真听有可能也是原因）。在前面神策数据与Ping++嘉宾分享的内容，个人认为很多还是和自己目前所看的一些相关的知识是一样的，并没有很大的创新点和学习内容，从AARRR模型、分析方法、第一关键指标法到复购魔法数字等，这些都可以说是目前自己所了解的，当然，并不是说这些没用，而是说，自己期望能够获得一些自己所不了解的内容，当然还有一部分原因是在于嘉宾分享的，更多的是与自己产品、业务联系紧密的一些内容，而对于这些，自己一直有着很大的困扰，那就是目前自己的业务和这个行业所讲述的Growth还是有着一定的区别的，没有能够很好的将其二者联系起来，至今不能说出明确的一二三。当然，最大的原因是自己并没有去实际认真考虑过这个问题。这也是引发自己思考的第二点。具体表现在于：1.只有输入没有输出，看过的学过的一些内容没有经过自己的消化和整理；2.最严重的在于一切都只是空想而没有实际行动。

嗯，总结来说，这个2017年下半年的第一天，自己的感触是以下几点，也是自己面临的几个很严重的问题：

  1. 对于自己没有一个清晰的认识和定位，对于未来没有规划和与之契合的行动；
  2. 只有空想没有实际行动；
  3. 只有输入没有输出，对于学到的东西没有自己的理解和消化

以上，还希望自己能够开始改进，对自己有一个认识和定位，一步步实践与行动起来，慢慢做好。
