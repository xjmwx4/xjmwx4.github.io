发表于 2017-05-04 | 分类于 数据分析

前言

目前正在考虑正式转向数据分析(技术型)方向，因此有在学习数据分析方面的硬技能如SQL/Python等。写这么一篇文章的目的是整理所学到的知识，对学到的知识点做一个基础性的梳理，加深自己理解的知识构建的同时也能之后做定期回顾以及知识点查询。

初衷是想做一个全面性的学习整理，因此这篇文章的目的是基础性、规范化、全面化，会在之后学习和实践的过程中不断完善，也不断地提升自己的知识水平和实践能力。

本篇是SQL篇。

几个基础性问题

在正式开始整理SQL的基础之前，有几个问题值得认真思考下：

  • 为什么要做数据分析？
  • 对于大数据行业的职业规划？
  • 为什么要学习SQL？
  • 具体学习哪种SQL语言？

以下是个人的回答
1.为什么要做数据分析？

有以下几个原因：
原因1：未来是一个信息与数据时代，大数据在科技领域、社会企业和社会生活方面扮演着尤为重要的角色；
原因2：目前从事的即为数据分析（初级），进一步从事技术型的数据分析，能够得到较好的职业发展，也能得到较为快速的行业入门；

2.对于大数据行业的职业规划？

在知乎有人对数据分析涉及的三个面做了一个说明，即分为BI (Business Intelligence), QA (Quantitative Analyst / Data Scientist), 以及BA (Business Analyst)三类。（详见如何快速成为数据分析师？——王曦的回答）

以上三个方向具体划描述与侧重为：
Business Analyst 定义问题，分析数据，提出并部分实现解决方案；
Quantitative Analyst 定义模型和架构，实现完整的解决方案；
Business Intelligence 对数据和度量进行可视化展示、自动化计算和更新。

3.为什么要学习SQL？
SQL是目前数据分析行业的一种通用性基础性语言，结构化查询的使用，使得快速处理大量/海量数据更加便捷…….

4.具体学习哪种SQL语言？
可以先参看下知乎这个回答：SQL Server 和 Oracle 以及 MySQL 有哪些区别？。考虑到自己想进驻的是互联网企业，同时从JD上来说，MySQL提及的次数相较SQL Server较多，因此目前选择学习的是MySQL。

SQL简介

SQL是结构化查询语言（Structured Query Language）的缩写。SQL是一种专门用来与数据库沟通的语言，设计SQL的目的是很好地完成一项任务——提供一种从数据库中读写数据的简单有效的方法。

SQL有如下的优点：

  • SQL不是某个特定数据库供应商专有的语言。几乎所有重要的DBMS都支持SQL，所以学习此语言使你几乎能与所有数据库打交道。
  • SQL简单易学。它的语句全都是由有很强描述性的英语单词组成，而且这些单词的数目不多。
  • SQL虽然看上去很简单，但实际上是一种强有力的语言，灵活使用其语言元素，可以进行非常复杂和高级的数据库操作。
